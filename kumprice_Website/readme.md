# KumPrice Website

A promotional website for the KumPrice mobile application - a capstone project for "Centralized Mobile Application for Local Entrepreneur Inventory Management and Consumer Price Viewing."

## Project Structure

```
kumprice_website/
├── index.html              # Main landing page
├── assets/                 # Static assets
│   ├── css/
│   │   └── styles.css      # Main stylesheet
│   ├── js/
│   │   └── script.js       # JavaScript functionality
│   └── img/
│       └── kumprice_content.jpg
├── pages/
│   └── manual.html         # User manual page
├── .github/
│   └── copilot-instructions.md
└── .well-known/
    └── assetlinks.json

```

## Features

- **Responsive Design**: Mobile-first approach with Bootstrap 5.3.2
- **Academic Focus**: Suitable for capstone project presentation
- **Terms & Conditions**: Legal compliance with modal popup before downloads
- **Modern UI**: Professional design with smooth interactions
- **Dual Audience**: Serves both SMEs and consumers

## Technologies Used

- HTML5 & CSS3
- Bootstrap 5.3.2
- Vanilla JavaScript
- Google Fonts (Plus Jakarta Sans)
- Font Awesome icons

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through sections or visit `pages/manual.html` for detailed documentation
3. All assets are properly linked and organized
